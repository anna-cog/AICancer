## Quick Start

Install necessary packages:

    pip install -r requirements.txt
    
Edit start.py to include histogram image path:

    ...
    img = 'dataset\demo\lungn10.png'
    ...
    
Run start.py:

    python start.py

Output:

    >>> Lung: Benign
